# UGUI-FlappyBird

## 项目
基于官方示例[Flappy Bird Style Example Game](https://www.assetstore.unity3d.com/#!/content/80330)，使用UGUI全面改造。

## 更新
* 增加分数越高，速度越快。
* 使用Shader更新背景贴图。
* 使用EventTrigger组件监控点击。
* 使用RectTransform组件移动柱子。
* 删除其他对象的刚体，仅保留玩家一个。
